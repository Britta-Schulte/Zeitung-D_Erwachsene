Die Angaben sind anonym. Dafür benutzen wir nicht deinen Namen, sondern einen „Code“.

Gib als Code bitte:

  • die ersten beiden Buchstaben des Vornamens deiner Mutter an,
  • den Tag deines Geburtsdatums
  • und die ersten beiden Buchstaben deines Vornamens.

    • Beispiel: AN08PE für Anna (Vorname deiner Mutter), 08. Januar 1999 (dein eigenes Geburtsdatum), Peter (dein eigener Name)

    Bitte den eigenen Code nach obigem Muster in das Textfeld eintragen.
